# Python 核心语法（从图片转写）

## 一、基础语法

### 1. 变量、字面值、类型
#### 1）什么是变量？
程序中用来存储单个数据的容器（经常会发生改变的数据）
```
num = 10
```

#### 2）变量的使用
- 输出打印
- 参与计算
- 记录数据

#### 3）注意事项
- 一个变量只能存储一个值
- 变量定义时必须赋值才可以使用
- 一条语句可以定义多个变量、也可以连续赋值
```
a,b = 1,"Python"
```

### 2）字面值
- 程序中，直接书写的固定值（数据）

#### 各类字面值书写形式
- 整数（int）/小数（float）：直接写，如 `10`、`3.14`
- 字符串（str）：双引号 `"人生苦短，我用Python"`
- 布尔（bool）：`True`、`False`
- 空值（NoneType）：`None`

### 3）数据类型
#### 常见数据类型
- `int`、`float`、`str`、`bool`、`NoneType`

#### 查看数据类型
- `type(...)`
- `isinstance(...)`

### 4）标识符命名规则
- 只能包含字母（a-z、A-Z）、数字（0-9）、下划线（_）
- **数字不能开头**
- 不能使用关键字：`if`、`else`、`elif`、`for`、`while`、`and`、`or` 等
- 严格区分大小写：`age`、`Age`、`AGE` 是三个变量

#### 变量命名规范
- 见名知意
- 多个单词使用下划线连接
- 英文字母全小写

### 5）运算符

#### 赋值运算符
- `=`：赋值，把右边结果赋值给左边，如 `num = 1+2`
- `+=`：加法赋值，`num +=2` 等价于 `num = num +2`
- `-=`：减法赋值，`num -=2`
- `*=`：乘法赋值，`num *=2`
- `/=`：除法赋值，`num /=2`
- `%=`：取模赋值，`num %=2`
- `//=`：取整除赋值，`num //=2`
- `**=`：幂赋值，`num **=2`

#### 比较运算符（关系运算符）
- `==`：等于，`a == b`
- `!=`：不等于，`a != b`
- `>`：大于，`a > b`
- `>=`：大于等于，`a >= b`
- `<`：小于，`a < b`
- `<=`：小于等于，`a <= b`

## 二、字符串

### 1）字符串特点
- 不可变性（无法修改）
- 有序性
- 可迭代性

### 2）字符串索引
- 正向索引：从0开始
- 反向索引：从-1开始

### 3）字符串切片
语法：`s[start:end:step]`
- `start`：起始索引，默认0
- `end`：结束索引，默认-1
- `step`：步长，默认1

### 4）字符串常用方法
- `find()`：查找子串，返回首次索引，找不到返回-1
- `count()`：统计子串出现次数
- `upper()`：转大写
- `lower()`：转小写
- `split()`：按分隔符分割成列表
- `strip()`：去除首尾空白/指定字符
- `replace()`：替换子串
- `startswith()`：判断开头，返回布尔

### 5）字符串拼接与格式化
#### 拼接
- 使用 `+`
- 只能拼接字符串，其他类型要转 `str(...)`

#### 格式化
- `%` 格式化：`"%s" % 变量`、`"%d" % 变量`
- `f-string`（推荐）：`f"内容{变量/表达式}"`

## 三、列表

### 1）列表定义与特点
- `列表名 = [元素1, 元素2, ...]`
- 可存放不同类型、可重复、**有序、可修改**

### 2）列表索引
- 正向：0开始
- 反向：-1开始

### 3）列表增删改查
- 查看：`list1[0]`
- 修改：`list1[0] = "A"`
- 删除：`del list1[3]`
- 注意：索引越界报错

### 4）列表常用方法
- `append()`：末尾追加
- `insert()`：指定位置插入
- `remove()`：移除第一个匹配值
- `pop()`：按索引删除（默认最后）
- `sort()`：排序（同类型）
- `reverse()`：反转

### 5）列表合并与成员判断
- 合并：`+`、`*` 解包
- 判断存在：`元素 in 列表`，返回布尔

### 6）数据统计
- `min()`：最小值
- `max()`：最大值
- `sum()`：求和
- `len()`：元素个数

## 四、元组

### 1）元组介绍
- 不可变序列，类似列表，**创建后不能修改**
- 可存不同类型、可重复、有序、不可修改

### 2）定义
- `元组名 = (元素1, 元素2, ...)`
- 空元组：`()`、`tuple()`

### 3）方法
- `count()`：统计次数
- `index()`：查找首次索引

### 4）组包与解包
- 组包：多个值合并到容器
- 解包：容器拆成独立变量
- `*`：收集剩余元素

## 五、字典

### 1）字典介绍
- 键值对（key:value）存储
- key 不可重复、可修改

### 2）定义
- `dict = {"name":"小王", "age":18}`
- 空字典：`{}`、`dict()`

### 3）取值
- `字典[key]`

### 4）常用操作
- 增加：`dict[key] = value`
- 删除：`pop(key)`、`del dict[key]`
- 修改：`dict[key] = value`
- 查询：
  - `get(key)`：取值
  - `keys()`：所有key
  - `values()`：所有value
  - `items()`：所有键值对

## 六、集合

### 1）集合介绍
- 无序、不可重复、可修改

### 2）定义
- `s = {"C","D","X","T","O","U"}`
- 空集合：`set()`（**不能用 `{}`**）

### 3）常用方法
- `add()`：添加元素
- `remove()`：移除指定元素（不存在报错）
- `pop()`：随机删除并返回
- `clear()`：清空
- `difference()`：差集
- `union()`：并集
- `intersection()`：交集

| 切片操作 | 支持     | 支持       | 支持     | 不支持 | 不支持     |
| 场景     | 文本处理 | 有序可重复 | 固定数据 | 去重   | 键值对     |

## 八、流程控制

### 1）if 条件判断
#### 基本格式
```
if 条件:
    代码
elif 条件:
    代码
else:
    代码
```

#### 注意事项
- 条件必须布尔类型
- 条件后必须冒号
- 代码块缩进（4空格）

### 2）match…case（Python3.10+）
```
match 表达式:
    case 值1:
        ...
    case 值2 if 条件:
        ...
    case 值3 | 值4:
        ...
    case _:
        ...
```
#### 场景
- 固定值分支：match
- 复杂逻辑：if/elif

### 3）while 循环
#### 格式
```
while 条件:
    循环体
else:
    ...
```

#### 注意
- 条件布尔
- 缩进
- 必须有终止条件，避免死循环

### 4）for 循环
#### 格式
```
for 元素 in 数据集:
    循环体
else:
    ...
```

#### for/while 场景
- while：次数未知、条件控制
- for：遍历已知集合、次数已知

### 5）range
- `range(end)`：0~end（不含end）
- `range(start,end)`：start~end
- `range(start,end,step)`：步长

## 九、函数

### 1）函数定义与调用
#### 定义
```
def 函数名(参数):
    """说明文档"""
    return 返回值
```

#### 调用
`函数名(参数)`

#### 注意
- 必须先定义后调用
- 定义不执行，调用才执行
- 缩进表示层级

### 2）参数类型

#### 位置参数
- 顺序必须一致

#### 关键字参数
- `形参名=值`，顺序不限
- 位置参数在前、关键字参数在后

#### 默认参数
- `def func(name, age=18)`
- 默认参数必须放在参数列表后面

### 3）变量作用域

#### 局部/全局变量
- 函数内：局部变量
- 函数外：全局变量

#### global
- `global x`：声明使用全局变量

#### 注意
- 尽量避免全局变量
- 优先用参数/返回值传递数据
- global 用于状态、配置、计数器

### 4）多返回值
- 打包成元组：`return a,b`
- 解包：`a,b = func()`

### 5）函数嵌套调用
- 一个函数里调用另一个
- 栈结构：后进先出（LIFO）

### 6）函数说明文档（Docstring）
- 三引号写在函数开头
- 说明功能、参数、返回值
- 查看：`help(函数名)`、IDE悬停
